# 🛍️ ELORIA – Modern E-Commerce Website

ELORIA is a modern, responsive, and user-friendly e-commerce website built using HTML, CSS, and JavaScript. The project focuses on delivering a smooth online shopping experience with an elegant design and interactive features.

## ✨ Features

- 🏠 Modern Landing Page
- 🛒 Shopping Cart Functionality
- ❤️ Wishlist Management
- 🌙 Dark Mode Toggle
- 📱 Fully Responsive Design
- 🔍 Product Showcase Section
- 🎨 Clean & Attractive User Interface
- ⚡ Interactive User Experience with JavaScript

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript
